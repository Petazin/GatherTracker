# GatherTracker

**GatherTracker** is a lightweight and efficient addon for World of Warcraft Classic that automatically toggles your minimap tracking (e.g., Minerals and Herbs) at configurable intervals. Perfect for gatherers who don't want to miss anything!

## Features

*   **Automatic Toggling:** Switches between two tracking types (Primary and Secondary Tracking) cyclically.
*   **Configurable Interval:** Adjust the switching speed from 2 seconds to 60 seconds.
*   **Smart Minimap Button:**
    *   **Left Click:** Toggle tracker On/Off.
    *   **Right Click:** Open configuration menu.
    *   **Mouse Wheel:** Quickly increase or decrease the time interval.
    *   **Alt + Drag:** Move the button anywhere on the screen.
*   **Smart Detection:** Automatically pauses if you are casting, in combat, or dead.

## Usage

1.  Once installed, you will see a button on your screen with your current tracking icon.
2.  **Left Click** the button to start toggling.
3.  Use **Mouse Wheel** on the button to adjust how fast the tracking changes.

### Chat Commands

*   `/gt` - Enables or disables the tracker.
*   `/gt opt` - Opens the full configuration panel.

## Configuration

In the options menu (Right Click on the button or `/gt opt`), you can:
*   Select which two resource types you want to track (e.g., Minerals and Herbs, or Treasure and Beasts for Dwarves/Hunters).
*   Hide the minimap button if you prefer using only keybinds.
